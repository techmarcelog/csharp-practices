# C# Practices

This repository contains my C# practice exercises and study materials focused on building a strong foundation for backend development with .NET.

It includes exercises and small practice projects covering programming logic, object-oriented programming, data structures, LINQ, async/await, and other essential C# concepts.

## Purpose

The goal of this repository is to document my learning journey in C# and organize practical exercises that help me improve my programming skills and prepare for backend development opportunities.

## Topics Covered

- Programming logic
- C# fundamentals
- Object-Oriented Programming (OOP)
- Classes and methods
- Encapsulation
- Inheritance and polymorphism
- Interfaces and abstraction
- Collections and list manipulation
- LINQ
- Async/Await
- Basic data structures
- Backend development fundamentals

## Repository Structure

The repository is organized by topic, making it easier to study and review specific concepts.

Example structure:

- `fundamentals/` → basic C# and logic exercises
- `oop/` → object-oriented programming exercises
- `linq/` → LINQ practice
- `async-await/` → asynchronous programming exercises
- `data-structures/` → stack, queue, linked list, and related topics

## Technologies Used

- C#
- .NET
- Visual Studio Code
